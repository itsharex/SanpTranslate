## 1. 产品概述

### 1.1 产品愿景
一款**极简、高效、隐私安全**的桌面截图贴图翻译工具。用户框选屏幕任意区域，截图自动贴在截取原位并存入剪贴板，点击按钮即可通过自备的大模型 API 完成翻译，译文直接覆盖原文上方或通过面板查看，体验无缝直观。

### 1.2 产品定位
- **形态**：系统托盘常驻工具，无臃肿主窗口，所有操作通过快捷键与桌面贴图完成。
- **核心价值**：原位截图、即时翻译、即时对照，零学习成本。
- **数据策略**：软件完全在本地处理截图，绝对不上传任何数据至第三方服务器（仅与用户自行配置的大模型 API 通信）。不扫描硬盘，不收集任何使用数据。

### 1.3 目标用户
- 需频繁阅读外语资料、论文、网页、软件界面的学生与研究者。
- 跨国协作中需快速理解外语文档、截图内容的职场人士。
- 对隐私高度敏感，拒绝截图数据被云端 OCR 服务获取的用户。

## 2. 功能需求

### 2.1 截图与贴图
- **截图快捷键**：全局快捷键 `Ctrl+Alt+L`（允许用户自定义）。
- **贴图快捷键**：`Ctrl+Alt+P`（从系统剪贴板取图贴到桌面，允许用户自定义）。
- **截图交互**：
  - 按下 `Ctrl+Alt+L` 后，屏幕覆盖半透明蒙版，鼠标变为精确选择状态。
  - 用户拖拽框选区域，松开鼠标完成截图。
  - `Esc` 键或鼠标右键随时取消。
- **截图后自动行为**：
  - 截图图片**自动以无边框置顶窗口的形式贴在屏幕原位置**（即用户框选的区域），完全覆盖原画面。
  - **同时将截图复制到系统剪贴板**，方便用户在其他应用中粘贴或通过贴图快捷键再次贴图。
- **从剪贴板贴图**：
  - 按下 `Ctrl+Alt+P`，自动读取当前剪贴板中的图像内容（若有），生成一张新的置顶贴图窗口。
  - 该贴图窗口**默认显示在屏幕中央**，可拖拽移动、双击关闭，后续行为与截图产生的贴图完全一致。
- **贴图通用行为**：
  - 所有贴图窗口均可**拖拽移动**到屏幕任意位置。
  - **双击贴图图片区域**即可关闭该贴图。右键菜单同时提供“关闭”选项作为辅助。
  - 多个贴图可并存，每个贴图独立操作。

### 2.2 翻译核心流程
翻译依赖用户自行配置的大模型 API，提供两种模式：

#### 模式 A：文字提取 + 纯文本模型翻译（OCR 模式）
- **本地文字提取**：内置离线文字识别能力，在本地提取图片中的文本及其精确位置坐标。
- **翻译构造**：
  - 将识别出的文本按顺序组织，发送至用户配置的**文本大模型 API**。
  - 指令要求模型返回与原文一一对应的译文，以便映射回各文本块。
- **适用场景**：需对多个独立文字区域进行原位覆盖翻译。

#### 模式 B：图像理解模型直接翻译（多模态模式）
- 直接将截图图像发送至用户配置的**多模态/视觉大模型 API**。
- 由模型一次性识别图中所有文字并输出翻译结果。
- **限制**：无法获得文字精确定位，结果为整体译文。

### 2.3 翻译结果展示与交互

#### OCR 模式结果
- 贴图窗口结构调整为：**上方为图片显示区，下方为透明背景控制栏**，按钮上边界贴合图片下边界，左边界与图片左边界对齐。
- 控制栏按钮：
  - **AI 翻译**（核心按钮，首次翻译前显示）
  - 翻译完成后显示 **“复制全部”**、**“原文/译文”切换**
  - （多模态模式下新增“译文面板”按钮，此处未激活时不显示或置灰）
- 翻译状态：点击翻译后，按钮变为“翻译中…”加载态。
- 翻译结果呈现：**每个原文文本块上方覆盖译文标签**。
  - 标签样式：**无圆角**，半透明深色背景，白色高对比文字。
  - 标签位置严格对齐文字提取坐标，译文过长时标签自动扩展，贴图窗口随之调整尺寸。
  - 原文被遮挡，通过“原文/译文”切换按钮可临时显示原文。
- **复制**：
  - **单击任一译文标签**，该段翻译复制到剪贴板，并短暂显示“已复制”。
  - 点击“复制全部”一键复制所有译文。

#### 多模态模式结果
- 翻译完成后，**不会自动弹出任何译文面板**。
- 控制栏出现 **“译文面板”按钮**，点击后**弹出可拖拽的浮动译文面板**，显示整体翻译文本。
- 面板支持用户自由关闭、再次打开，文本可自由选择与复制。
- 面板内提供“合并到贴图”选项，可将译文以半透明样式覆盖在图片上。

### 2.4 API 配置管理
- **入口**：系统托盘右键菜单 → “设置”。
- **需配置项**：
  - API 基础地址
  - API 密钥（安全输入，可切换显示/隐藏）
  - 文本翻译模型名称
  - 图像理解模型名称（留空则仅使用 OCR 模式）
  - 目标翻译语言（下拉列表）
  - 默认翻译模式（OCR 优先 / 多模态优先）
- **安全存储**：密钥使用操作系统提供的安全存储机制加密保存，配置文件本地加密。
- **连接测试**：提供“测试连接”按钮，发送最小请求验证可用性，并反馈成功或失败。

### 2.5 系统托盘与快捷键
- **托盘图标**：软件启动后常驻系统托盘。
- **托盘菜单项**：
  - 框选截图翻译（`Ctrl+Alt+L`）
  - 从剪贴板贴图（`Ctrl+Alt+P`）
  - 翻译最近一张贴图
  - **截图与翻译历史**（打开历史记录面板）
  - 设置
  - 退出
- **快捷键**：所有全局快捷键（`Ctrl+Alt+L`、`Ctrl+Alt+P` 等）允许用户自定义。

### 2.6 界面语言
- 软件自身界面文字跟随操作系统语言自动切换（最低支持中文、英文）。

### 2.7 截图与翻译历史
- **功能目标**：让用户回顾过去一段时间的截图及对应翻译，方便复用。
- **历史记录内容**：
  - 每次通过 `Ctrl+Alt+L` 截图并完成翻译（OCR 模式识别并翻译，或多模态模式获得译文），自动保存历史项：截图缩略图、OCR 识别文本（如有）、翻译后文本、翻译模式、时间戳。
  - 仅存储缩略图与文本，不保存完整原图以节省空间。
- **查看与管理**：
  - 通过托盘菜单“截图与翻译历史”打开历史面板。
  - 面板以列表形式展示最近的历史条目（如最近 50 条），显示缩略图与翻译摘要。
  - 点击某项可查看详细信息、复制翻译文本。
  - 支持**逐条删除**或**一键清空全部历史**。
  - **所有数据完全本地存储，绝不联网**，用户可随时彻底清除。
- **补充说明**：通过 `Ctrl+Alt+P` 单纯贴出的图片若未进行翻译，不会自动进入历史。

## 3. 非功能需求

### 3.1 性能
- 截图响应延迟 < 100 毫秒。
- 常规尺寸截图本地文字提取 < 0.5 秒。
- 多贴图并存时内存占用合理，不引起系统卡顿。

### 3.2 安全性
- 截图数据仅存在于内存，不写入磁盘（历史记录缩略图与文本仍完全本地存储）。
- 与用户配置 API 的通信加密，严格验证证书。
- 绝不扫描、读取硬盘无关文件。
- 无任何后台遥测、崩溃上报。

### 3.3 易用性
- 首次未配置 API 时自然引导至设置。
- 贴图原位显示，双击即关，截图自动加入剪贴板，符合截图工具通用习惯。
- `Ctrl+Alt+P` 一键贴出剪贴板图片，操作直观。
- 按钮集中在图片下方，透明背景不遮挡桌面内容，操作便利。
- 标签直角设计，简洁利落。

### 3.4 跨平台兼容
- 支持 Windows 10+、macOS 12+、主流 Linux 发行版（兼容 Wayland/X11）。
- 安装包轻量，包含离线文字识别所需资源后体积合理。

### 3.5 可维护性
- 离线文字识别能力可独立更新。
- 翻译调用与界面展示低耦合，适配不同 API 风格。
- 提供可开启的本地运行日志，方便排查问题。

## 4. 用户界面与体验需求

### 4.1 设置界面
- 极简表单布局，仅必要字段。
- 密钥字段默认掩码，测试连接后给出明确状态提示。

### 4.2 贴图窗口
- **结构**：
  - 上部：截图图片显示区，保持原始清晰度。
  - 下部：**透明背景控制栏**，按钮上边界贴合图片下边界，左边界与图片左边界对齐。包含所有操作按钮（从左至右）：AI翻译/翻译中/原文切换、复制全部（翻译后显示）、译文面板（多模态模式下翻译后显示）。
  - 窗口无标题栏，无传统关闭按钮（主关闭方式为双击图片）。
- **交互**：
  - 拖拽图片区域或控制栏空白处可移动窗口。
  - **双击图片区域关闭贴图**。
  - 控制栏按钮状态随翻译进度动态变化。

### 4.3 翻译覆盖层（OCR 模式）
- 每个译文标签为独立浮层，**无圆角（直角）**，半透明深色背景，白色文字。
- 鼠标悬停变手形，单击复制并浮现“已复制”提示。
- 标签位置匹配文字坐标，允许扩大，贴图窗口随之扩展。
- 切换原文时所有标签隐藏，图片原文完整可见。

### 4.4 译文面板（多模态模式）
- 用户主动触发展示，非自动弹出。
- 面板为浮动窗口，可拖拽，包含整体译文文本。
- 文本可选择、复制，提供“合并到贴图”按钮。
- 关闭面板可随时通过面板右上角按钮或再次点击控制栏“译文面板”按钮。

### 4.5 历史记录面板
- 独立窗口，列表形式展示缩略图、翻译摘要及时间。
- 支持复制翻译、删除单条、清空全部。
- 界面极简，符合整体风格。

## 5. 功能数据流

### 5.1 截图及贴图流程
1. 用户按下 `Ctrl+Alt+L` → 框选区域 → 松开鼠标完成截图。
2. 截图图像自动生成置顶贴图在原位，同时复制到系统剪贴板。
3. 用户可在贴图上进行翻译操作，或按 `Ctrl+Alt+P` 将剪贴板中的同一图像或其他图像贴为新窗口（默认居中）。

### 5.2 OCR 翻译模式流程
1. 在贴图上点击“AI翻译” → 图像送入本地文字提取 → 得到文本及坐标。
2. 组织文本发送至文本大模型 API → 获取对应译文。
3. 依据坐标在图片上方覆盖直角译文标签。
4. 用户可单击标签复制、切换原文、复制全部。

### 5.3 多模态翻译模式流程
1. 在贴图上点击“AI翻译” → 图像发送至视觉大模型 → 返回整体翻译。
2. 控制栏出现“译文面板”按钮，用户点击后打开浮动面板查看/复制译文，可合并到图。

### 5.4 历史记录流程
- 通过 `Ctrl+Alt+L` 截图并完成翻译后，自动将截图缩略图、原文文本、译文、时间戳记入本地历史数据库。
- 用户通过托盘菜单进入历史面板查看管理。

## 6. 验收标准

- [ ] 全局快捷键 `Ctrl+Alt+L` 可正常框选截图，截图后贴图在原位且图像已入剪贴板。
- [ ] 快捷键 `Ctrl+Alt+P` 可将剪贴板中的图片贴到桌面（无剪贴板图像时不做响应或以提示反馈）。
- [ ] 贴图可拖拽移动，双击关闭，控制栏按钮集中在下部，无圆角标签样式符合描述。
- [ ] OCR 模式翻译覆盖准确，单击标签复制，切换原文/译文正常。
- [ ] 多模态模式翻译后不自动弹面板，手动触发展示、复制与合并。
- [ ] 历史记录随截图翻译自动生成，可查看、复制、删除、清空，`Ctrl+Alt+P` 贴图不产生历史。
- [ ] API 配置加密存储可靠，测试连接反馈明确。
- [ ] 无任何数据外传行为，性能与跨平台表现符合要求。
